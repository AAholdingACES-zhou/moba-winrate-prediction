# 基于神经网络模型的 5v5 MOBA 游戏胜率预测

本项目通过对给定 MOBA 游戏对局数据进行清洗、分析与建模，对比了 **逻辑回归模型** 与 **全连接神经网络模型（ANN）** 的预测效果，并最终利用性能更优的神经网络输出对局胜率预测结果。

主要技术路线包括：数据预处理、特征可视化、模型训练、参数调优以及 ROC 曲线绘制。

---

## 📌 项目概述

本研究的主要步骤如下：

- 数据清洗与缺失值处理  
- 异常值修复（例如英雄 ID 异常、未按 5v5 规则选取等）  
- 删除从未被选择过的英雄特征  
- 数据可视化（英雄选择频率、队伍分布、胜负场次等，可见论文第 7–15 页图表展示）
- 构建并比较两类监督学习模型  
- 输出预测概率与 ROC 曲线评价指标

使用的 Python 库包括：  
`numpy`、`pandas`、`matplotlib`、`seaborn`、`sklearn`、`tensorflow.keras`。

---

## 🧹 1. 数据预处理

对原始数据进行如下处理：

- 检查空值并补全  
- 修复异常值（如异常的 100 值）  
- 检查每行数据是否满足“5v5 英雄选择规则”  
- 删除从未被选取过的英雄列  
- 使用 `StandardScaler` 对特征进行标准化处理  

这些操作保证了模型训练的有效性。

## 🤖 3. 模型构建与对比

### **（1）逻辑回归模型（Logistic Regression）**
由于预测目标为二分类（胜 / 负），首先尝试逻辑回归。

- 基础模型 AUC ≈ **0.596**  
- 调整 solver 与正则化参数后，AUC 仍无明显提升（≈ 0.596）

逻辑回归难以捕捉多英雄组合的复杂非线性关系，因此效果有限。

---

### **（2）全连接神经网络模型（ANN）**

构建了两种神经网络结构并进行训练：

#### 🟦 **模型 1（简单结构）**
- Dense(16) → Dense(2)  
- AUC ≈ **0.657**

#### 🟧 **模型 2（调参后）**
- Dense(32) → Dense(64) → Dense(2)  
- 更大的 batch size 与更多 epochs  
- AUC ≈ **0.737**（最佳结果）

比逻辑回归显著提高预测性能。

论文中的 ROC 曲线（图 9，页码 15）展示了两种网络结构的性能差异 :contentReference[oaicite:3]{index=3}。

---

## 📈 4. 最终结果

- **逻辑回归：AUC ~ 0.59（效果一般）**  
- **神经网络：AUC ~ 0.74（效果显著提升）**

最终使用调参后的神经网络模型输出测试集的胜率预测值。
